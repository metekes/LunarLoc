# Lunar Lander-Aided Visual-Inertial Localization Dataset

This repository contains the dataset accompanying the paper:

> **LAL-VINS: Lander-Anchored Visual-Inertial Navigation for Lunar Rovers**

---

## Download

The dataset is available through Google Drive:
* **Dataset:** [Google Drive Folder](https://drive.google.com/drive/folders/1Lh36zjT1xNvxIlwV56p27IdocCGfiEXD) 

## Status

**This repository is under active development.**

* Detailed documentation describing the dataset structure, file formats, and coordinate frames will be added in future updates.
* Additional sequences will also be released in future versions of the dataset.
